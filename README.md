## 特色

- **函数即武器**：输入 `sin(x)`、`x^2`、`tan(x)` 或任意合法的数学表达式，轨迹即为弹道
- **函数预设库**：内置常用曲线（台阶、高斯、Sigmoid、抛物线等），支持参数实时调节与预览
- **弹道预测**：输入函数时实时渲染预测轨迹，所见即所得
- **坐标/玩家坐标显示**：可开关的参考坐标系与玩家坐标信息
- **输入隐藏模式**：隐藏你的函数文本，只在网络上传输扩展式，防止对手发现你作弊

## `Plus` 模块说明

`src/Graphwar/Plus.java` 中集中管理了客户端的扩展开关。如果你想自定义功能的默认值，可修改以下字段后重新构建：

```java
public static final boolean canJoinGlobal = false;  // 是否允许连入公共大厅
public static boolean SHOW_COORDINATE_SYSTEM  = true; // 显示坐标系
public static boolean SHOW_PLAYER_COORDINATES = true; // 显示玩家坐标
public static boolean PREDICTION_ENABLED       = true; // 开启弹道预测
public static boolean hideTyping               = true; // 隐藏函数输入
```

> 注意：原版本的公共大厅（`www.graphwar.com:23761`）是否可连通取决于该服务器的当前状态。`canJoinGlobal = false` 时使用默认 Global 地址会被拦截并显示「此版本不允许加入Global服务器」——但你手动改为其他 IP/端口后仍可连接。

## 常见问题

**Q: 连接超时 / "connect timed out"**

- 确认目标服务器仍在运行。

**Q: 连入自己的私服 / "臭外挂"字样**

- 可能是首次连接失败但 `gameData` 仍残留状态。重启游戏即可。

**Q: 中文乱码**

- 自己修。编码问题。
